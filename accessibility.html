Accessibility statement & keyboard shortcuts

Statement

GMU Coding Club is committed to providing an accessible web experience. This static demo includes semantic HTML, focus styles, and keyboard support for interactive components.

Keyboard shortcuts

  • Alt + L: Open/close login modal
  • Esc: Close open modal
  • Tab / Shift+Tab: Navigate within modal when open

For full production sites, run accessibility audits (axe, Lighthouse) and test with assistive tech.
